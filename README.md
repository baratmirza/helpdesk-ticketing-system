# helpdesk-ticketing-system
This project was created to simulate a basic IT HelpDesk environment and demonstrate
how ticketing systems are used to manage and resolve user issues in a structured way.

Tools Used
-osTicket
-Windows OS

Features / What I Did
-Set up a ticketing system environment
-Created sample user requests (login issues, system errors, printer problems)
-Managed tickets from creation to resolution
-Categorized and prioritized incidents
-Simulated real IT support workflow

Outcome
This project helped me understand:
-How HelpDesk systems work in real environments
-Ticket lifecycle (open → in progress → resolved)
-Basic IT support and troubleshooting process
